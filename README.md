# FVS Incorporações - Site Institucional

Site institucional moderno para FVS Incorporações, desenvolvido com Next.js 14+, TypeScript e Tailwind CSS.

## 📋 Informações do Projeto

- **Empresa**: FVS Incorporações
- **Objetivo**: Apresentação de projetos, valores e geração de leads
- **Status**: 🔨 Em desenvolvimento (Fase 1 - Arquitetura)
- **Stack**: Next.js 14+, TypeScript, React, Tailwind CSS

## 📁 Estrutura de Diretórios

```
.
├── app/                    # Next.js App Router
├── components/             # Componentes React
│   ├── ui/                # Componentes base
│   ├── layout/            # Header, Footer, etc
│   └── sections/          # Seções de página
├── lib/                   # Utilitários e helpers
├── public/                # Assets estáticos
├── styles/                # Estilos globais
├── docs/                  # Documentação técnica
├── CLAUDE.md             # Instruções para Claude Code
├── FVS_MASTER_BUILD_SPEC_v2.md  # Build specification
└── README.md             # Este arquivo
```
